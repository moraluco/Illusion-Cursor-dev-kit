# 知识库

本目录用于沉淀研发过程中的概念、决策、易错点与参考资料，便于后续查阅和给 AI 提供上下文。阶段与任务拆解见 [../plan.md](../plan.md)，设计见 [../design/](../design/)，开发说明见 [../dev/](../dev/)。

## 使用方式

- 研发过程中把关键概念、链接、决策与易错点整理到对应文件。
- 在 Cursor 中 @ 引用这些文件，可让 AI 基于本地知识库回答。
- 本目录约定可同步至团队知识库或协作平台，同步方式由项目自定。
