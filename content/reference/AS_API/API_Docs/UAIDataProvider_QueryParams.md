# UAIDataProvider_QueryParams

**Visibility:** public

## Inheritance

UAIDataProvider → UAIDataProvider_QueryParams → UAIDataProvider_Random

## Description

AIDataProvider_QueryParams is used with environment queries

It allows defining simple parameters for running query, which are not tied to any specific pawn, but defined for every query execution.

### ParamName
