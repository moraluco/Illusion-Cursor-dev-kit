# UAudioBus

**Visibility:** public

## Inheritance

UObject → UAudioBus

## Description

An audio bus is an object which represents an audio patch cord. Audio can be sent to it. It can be sonified using USoundSourceBuses.  Instances of the audio bus are created in the audio engine.
