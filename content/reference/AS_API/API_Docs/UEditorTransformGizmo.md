# UEditorTransformGizmo

**Visibility:** public

## Inheritance

UInteractiveGizmo → UEditorTransformGizmo

## Description

UEditorTransformGizmo provides standard Transformation Gizmo interactions, applied to a UTransformProxy target object. By default the Gizmo will be a standard XYZ translate/rotate Gizmo (axis and plane translation).
