# UVirtualTexture2D

**Visibility:** public

## Inheritance

UTexture2D → UVirtualTexture2D

## Description

Virtual Texture with locally configurable build settings.  A raw UTexture2D can also represent a Virtual Texture but uses the one and only per-project build settings.
