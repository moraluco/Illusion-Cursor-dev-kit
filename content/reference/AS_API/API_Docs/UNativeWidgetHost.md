# UNativeWidgetHost

**Visibility:** public

## Inheritance

UWidget → UNativeWidgetHost

## Description

A NativeWidgetHost is a container widget that can contain one child slate widget.  This should be used when all you need is to nest a native widget inside a UMG widget.
