# UCameraModifier_CameraShake

**Visibility:** public

## Inheritance

UCameraModifier → UCameraModifier_CameraShake

## Description

A UCameraModifier_CameraShake is a camera modifier that can apply a UCameraShakeBase to the owning camera.

## CAMERAMODIFIER_CAMERASHAKE

### SplitScreenShakeScale

```
float32 SplitScreenShakeScale
```

Scaling factor applied to all camera shakes in when in splitscreen mode. Normally used to reduce shaking, since shakes feel more intense in a smaller viewport.
