# UGizmoArrowObject

**Visibility:** public

## Inheritance

UGizmoBaseObject → UGizmoArrowObject

## Description

Simple arrow object intended to be used as part of 3D Gizmos.  Draws a solid 3D arrow based with a cylinder body and cone or box head.

## OPTIONS

### Direction

```
FVector Direction Offset float32 Offset
```

Arrow origin is located at (Direction * Offset)
