# UCreateMeshObjectTypeProperties

**Visibility:** public

## Inheritance

## Description

UCreateMeshObjectTypeProperties is a InteractiveTool PropertySet used to select what type of object to create, in creation tools (ie StaticMesh, Volume, etc).
