## Description

ACTOR RECORDING bRecordTransforms bool bRecordTransforms

Whether to record actor transforms. This can be useful if you want the actors to end up in specific locations after the sequence.  By default we rely on animations to provide transforms, but this can be changed using the "Record In World Space" animation setting.
